5.1  Мета роботи
   На лабораторній роботі №5 потрібно розгорнути програмну систему та продемонструвати її роботу.
   5.2 Хід роботи
1) Розгорнути реалізовану програмну систему.
Програмну систему було розгорнуто завдяки Microsoft Azure що забезпечує гнучкість та масштабованість. Створено ресурсну групу в Azure. Налаштовано App Service для хостингу .NET 7.0. Додано базу даних SQL Server на хостингу. 
2)  Перевірити та налаштувати роботу розгорнутої програмної системи.
Виконано тестові HTTP-запити через REST API для перевірки CRUD-операцій. Перевірено отримання даних від IoT-клієнта та бази даних. Використано Azure Log Stream для моніторингу запитів. Виявлено проблему з FileNotFoundException, усунуто шляхом коригування NuGet-залежностей. Додано автошкалювання ресурсів у Azure App Service, оптимізовано запити до бази даних. Перевірено логіку для моніторингу стану доріг у реальному часі.
3) Продемонструвати описану у розділі 2.2 Vision & Scope функціональність програмної системи.
Демонстрація функціональності програмної системи була викладена у відеозаписі на YouTube (див. Додаток А).
